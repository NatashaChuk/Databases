2.	Explain when a database would be used. Explain when a text file would be used.

		A:  When data needs to be update, manipulated or saved by one or more users, a database should be used. When the data is static, a text file should be used.


3.	Describe one difference between SQL and other programming languages.
		
		A: SQL is a declarative language rather than a procedural one, meaning we only tell it what we're looking for versus how to look for it.


4.	In your own words, explain how the pieces of a database system fit together at a high level.
		
		A: A database system is similar to a spreadsheet. There are rows and columns of data that can be searched via an algorithm.

5.	Explain the meaning of table, row, column, and value.
		
		A:	Table - a set of columns and rows containing data - format for displaying data
		   	Row - individual pieces of data that make up a complete conceptual unit
			Column - defines what the data represents
			Value - a piece of information that the column name describes
